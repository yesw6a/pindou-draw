# 📱 移动端 H5 适配说明

## ✅ 已适配功能

### 1. Viewport 优化
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
```
- 禁止缩放
- 自适应屏幕宽度
- 触摸优化

### 2. 移动端工具栏
- 底部工具栏（手机模式）
- 常用工具快速访问
- 画笔/油漆桶/吸管/选区

### 3. 响应式布局
- CSS 媒体查询：`@media (max-width: 767px)`
- 自动切换手机/平板模式
- 弹窗自适应屏幕

### 4. 触摸支持
- 触摸绘制支持
- 手势优化
- 防止默认触摸行为

### 5. 平板模式优化
- `body.tablet-mode` 样式适配
- 弹窗尺寸调整
- 工具栏布局优化
