更新说明

贺词

历经将近两个月的努力，拼豆草图制作网页 V3.0 正式上线！ 本次更新主要优化响应式，以及功能上的优化，带来更好的使用体验以及更完善的操作。

一、全面适配平板端

网页现已支持平板设备，针对触控操作进行了特别优化。

当处于平板端时，可在 左侧工具栏 -> 使用说明与更新日志 -> 使用说明 查看平板端说明书(点击此处快捷跳转)

若使用平板时仍为电脑端，则在 左侧工具栏 -> 全屏与简洁模式 -> 切换为平板端显示与操作 可手动切换。

建议使用方式：在左侧工具栏开启全屏模式，获得最佳创作视野。

注：移动设备暂未支持手机设备。

二、“以图生草图”功能升级

  • 色彩表现优化

    改进了色卡匹配逻辑，有效避免生成结果颜色过暗的问题。

  • 透明背景处理优化

    上传透明背景图片时，透明区域将不再被填充，自动保留为空白，让草图生成更符合预期。

三、本地存储

现已支持在浏览器临时存储画布，且每五分钟或出发误退出提醒时会自动保存，有效避免数据丢失问题。

注：本地存储仅用于临时存储，如需长期保存请导出为PD工程文件。

四、初次使用提示框

现在对于新用户会有一个基础的使用小提示框，尽量避免初次使用困难。

五、其他优化

  • 修正了部分文本句不对意的问题。
  • 现在可以将初始弹窗(更新公告、初始使用教程)选择此后不再显示。
  • 优化了以图生草图的操作界面。

六、BUG修复

  • 修复了画布有时候会突然卡死的问题。
  • 修复了批量导出高亮图时，有时不会导出特殊色的问题。
  • 修复了以图生草图会使用珠光色作为初始色的问题。
  • 修复了画布光效有时不会实时改变的问题。
  • 修复了导入 PD 文件时色卡不会自动切换的问题。
  • 修复了参考窗口中的图片有时无法作为底图的问题。
  • 修复了底图有时无法缩放的问题。
  • 修复了因 CSS 类名书写错误导致样式异常的问题。

感谢各位粉丝和使用者的反馈！

这是什么网页

解答

该网页用于拼豆手工草图绘制，不支持专业的像素图绘制。

平台切换

帮助

网页支持电脑设备和平板设备使用，暂不支持手机设备使用。

  • 平板使用的依旧是电脑怎么办？

    支持手动切换 左侧工具栏 -> 全屏和简洁模式开关(倒数第二个选项) -> 改为平板端操作与显示

    特征：常态模式下有 [左右下] 三个工具栏，且右侧基础工具有橡皮模式选择

  • 电脑使用的依旧是平板怎么办？

    支持手动切换 左侧工具栏 -> 全屏和简洁模式开关(倒数第二个选项) -> 改为电脑端操作与显示

    特征：常态模式下有 [左右] 两个工具栏，且右侧基础工具没有橡皮模式选择
